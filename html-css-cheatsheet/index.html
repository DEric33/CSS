Sélecteurs CSS

Selecteurs simples Syntaxe Exemple Objectif
de type nomElement input Cibler n'importe quel élément <input>.
de classe .nomClasse .index Cibler n'importe quel élément qui possède la classe index
d’identifiant #valeurId #toc Cibler l'élément qui possède l'identifiant toc
universel * * Cibler tous les éléments du document
d’attribut [attr] [autoplay] Cibler tous les éléments qui possède l'attribut autoplay défini
Combinateurs Syntaxe Exemple Objectif
de voisin direct A + B div + p Cibler n'importe quel élément <p> qui suit immédiatement un élément <div>.
de voisins A ~ B p ~ span Cibler les éléments <span> qui suivent (immédiatement ou non) un élément <p> et qui ont le même élément parent.
d’éléments enfants A > B ul > li Cibler tous les éléments <li> qui sont directement situés sous un élément <ul>.
d’éléments descendants A B div span Cibler n'importe quel élément <span> situé à l'intérieur d'un élément <div>.
de colonne A || B col || td Cibler n'importe quel élément <td> sous la portée d'une colonne <col>.
Pseudo Syntaxe Exemple Objectif
Pseudo-classes a:visited Cibler l'ensemble des éléments <a> (des liens) ayant déjà été visités par l'utilisateur.
Pseudo-elements p::first-line Cibler la première ligne de chacun des éléments <p> (les paragraphes) du document.
